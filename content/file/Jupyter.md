#uni 
Estensione Jupyter Notebook.

Le celle di codice vengono eseguite su un "kernel".
Viene mantenuta una sessione globale.

Cell types:
- code
- markdown
- raw cells

Magic commands: non vengono passate al kernel
cell magics:
- `%%time`
- `%%HTML`
- `%%bash`
line magics:
- `%timeit`
- `%pwd`
- `%matplotlib inline`
