#uni 
The two most important application paradigms are:
- [[Peer to peer]]
	- bitTorrent
- [[Client-Server]]
	- web, http
	- e-mail, smtp, imap
	- dns

Le applicazioni distribuite girano su almeno due computer: il client e il server oppure N _peers_.
Le applicazioni girano solo sulla periferia, ovvero sugli Host (in realtà anche sui Router).
I router si limitano solo ad installare datagrams quindi non serve che implementino tutto il protocollo.
La rete è trasparente, nel senso che apparentemente le due macchine comunicano direttamente tra di loro.
